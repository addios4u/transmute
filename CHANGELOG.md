# Changelog

All notable changes to the **Transmute** extension will be documented in this file.

## [1.0.0] - 2025

### Added

- Initial release with 18 developer utility tools
- **Base64**: Image → Base64, Base64 → Image
- **JSON**: JSON Formatter, JSON → TypeScript, JSON → XML, XML → JSON
- **Image**: Image Converter, Image Resizer
- **PDF**: PDF Merge, PDF Split
- **Utility**: URL Encoder/Decoder, Hash Generator, UUID Generator, Timestamp Converter
- **Developer**: JWT Decoder, Regex Tester, Color Converter, CSV ↔ JSON
- Monaco Editor integration for code editing tools
- Multi-language support: English, Korean, Japanese, Chinese
- Full offline processing — no data leaves your device
- Clipboard support (copy, cut, paste, select all)
